# long-s-converter
### [Click here to use the converter.](https://travisgk.github.io/long-s-converter/)
This online tool can be used to **accurately** insert the historical long S character (&nbsp;Å¿&nbsp;) into text to make it appear as it would two centuries ago.

English, French, German, Spanish, and Italian are supported.
